# Tentacles Telegram Bot

Многофункциональный Telegram-бот на Python (aiogram 3) с тремя режимами работы.

## Режимы

### 1. AI Ассистент (`/change 1`)
- **Чат** — GPT-4o, GPT-4o Mini, Claude Sonnet/Haiku 4, Gemini 2.0/2.5 Flash (`/chat`)
- **Генерация изображений** — модель `gpt-image-1`, качество low/medium/high (`/image`)
- **Генерация видео** — Sora 2 / Sora 2 Pro, размеры до 1792x1024, длительность 4/8/12 сек (`/gen`, `/genref`)
- **Озвучка (TTS)** — модели `tts-1` / `tts-1-hd`, 9 голосов (`/tts`)
- **Распознавание речи (STT)** — отправка голосового сообщения
- **Управление API-ключами** — персональные и глобальный пул ключей Emergent (`/key`)

### 2. Видео инструменты (`/change 2`)
- **Склейка видео** — отправка нескольких видео, нормализация в 1280x720, склейка с опциональным движущимся водяным знаком (`/merge`, `/merge_now`)
- **Скачивание видео** — через yt-dlp, поддержка 1000+ сайтов (`/download`)
- **Сжатие видео** — два метода:
  - **FFmpeg** — кодеки h264/h265/av1, задание целевого размера в МБ (`/compress URL размер [кодек]`)
  - **ezgif.com** — онлайн-сжатие с выбором разрешения, битрейта и формата
- Результаты загружаются на **tempshare.su**

### 3. Перезалив файлов (`/change 3`)
- Скачивание файла по прямой ссылке и загрузка на **gigafile.nu**
- Поддержка файлов до 300 ГБ, хранение 100 дней
- Memory-safe: чанковая загрузка (50 МБ), потоковое скачивание

## Структура файлов

| Файл | Описание |
|---|---|
| `bot.py` | Точка входа, главное меню, маршрутизация по режимам, fallback-обработчики |
| `config.py` | Конфигурация бота, константы, управление режимами/настройками/ключами, утилиты форматирования |
| `ai_handlers.py` | Роутер AI-режима: чат, генерация изображений/видео, TTS, STT, настройки, ключи |
| `video_handlers.py` | Роутер видео-режима: склейка, скачивание (yt-dlp), сжатие (FFmpeg + ezgif) |
| `file_handlers.py` | Роутер режима перезалива файлов (URL -> GigaFile.nu) |
| `compressor.py` | FFmpeg-компрессор с сегментацией, автоподбором битрейта и повторными попытками |
| `gigafile_client.py` | Async-клиент GigaFile.nu — memory-safe чанковая загрузка до 300 ГБ |
| `recovery.sh` | Bash-скрипт восстановления: venv, зависимости, ffmpeg, supervisor |
| `requirements.txt` | Зависимости Python (130 пакетов) |
| `.env` | Переменная `TELEGRAM_BOT_TOKEN` |

## Технологии

- **Python 3.12+**, **aiogram 3.26** (async Telegram Bot API)
- **FFmpeg / ffprobe** — обработка видео
- **yt-dlp** — скачивание видео с 1000+ сайтов
- **emergentintegrations** — единый SDK для OpenAI (видео, изображения, TTS, STT, чат), Anthropic, Gemini
- **aiohttp / httpx** — асинхронные HTTP-запросы
- **beautifulsoup4** — парсинг ezgif.com
- **supervisor** — управление процессом бота
